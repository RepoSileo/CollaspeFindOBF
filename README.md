# CollapseFindOBF
Collapse find pls obf this file
| Status build: Work

## Installation

### From Source

```bash
git clone https://github.com/RepoSileo/CollapseFindOBF.git
```

```bash
cargo build --release --features gui --no-default-features
```

The binary will be available at CollapseScanner\target\release

### From Releases

Download the latest release from the [releases page](https://github.com/RepoSileo/CollapseFindOBF/releases).
